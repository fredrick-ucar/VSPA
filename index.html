Security Questionnaires

The following questionnaires below are used by UCAR to assess the security programs of third parties.

NOTE: Please only fill out the questionnaires that are applicable to the application or service you are providing. If your application is Software as a Service (SaaS), please fill out all of the questionnaires.

The questionnaires cover the following topics:

  • Web Application Security Questionnaire
  • Security & Privacy Program Questionnaire
  • Infrastructure Security Questionnaire
  • Physical & Datacenter Security Questionnaire
  • Cloud Application Security Questionnaire

Instructions

Complete the following steps for each security questionnaire link above.

  1. Click each questionnaire link and answer the questions. Some answers may require more explanation depending on the response.
  2. When done answering questions, click the Save buton at the bottom of the questionnaire to export the answers to a text file.
  3. Send an email to fredrick@ucar.edu and attach the exported text file to the email.
  4. The reviewer will open the questionnaire and load the answers received via email.
